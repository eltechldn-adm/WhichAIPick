# WhichAIPick — Master Control Document (MCD)
Status: LOCKED (Only El can change this)

---

## 1) Product Purpose (LOCKED)
WhichAIPick helps people discover AI tools they didn’t know existed and find the right AI tool for their specific need.

## 2) Primary Monetisation (LOCKED)
Traffic + affiliate revenue.
All UX and content decisions must support discovery, browsing depth, and click-through to tool links.

## 3) Audience (LOCKED)
For: AI-curious users, builders, creators, operators, solopreneurs.
Not for: People who hate AI.

## 4) Desired User Feeling (LOCKED)
Curiosity + Power.

## 5) Brand Feel (LOCKED)
Cinematic + Technical.
Premium, serious, clean, visually attractive.

## 6) Visual Bans (LOCKED)
- No neon
- No blue or purple tones
- No cartoon AI style
- No stock-photo SaaS vibe
- No cheesy “AI brain glow” clichés (unless explicitly approved by El)

## 7) Colour Palette (LOCKED)
Primary background: #0F1215 (Charcoal Black)
Accent: #F5A623 (Warm Amber)
Text: #F4F4F4 (Soft Off-White)
Panels/secondary: #2A2E33 (Steel Grey)

## 8) Logo Rules (LOCKED)
- Icon-first
- Placement: Header and Footer only
- Immutable: No regeneration, recolouring, reshaping, or animation
- The agent must never modify logo assets

Authoritative file path:
- /assets/logo/logo-final.png

## 9) Hero Rules (LOCKED)
Home page only.
Full cinematic video background.
Purpose: scale + discovery + calm power.
Motion: slow, controlled, loopable (8–12s).
Must NOT include: text, logos, UI mockups, charts, dashboards, robots, faces, cities, HUD overlays, blue/purple, neon, glitch chaos.
Agent must never regenerate or alter hero video.

Authoritative file path:
- /assets/hero/hero-video.mp4

## 10) Global Page Visual Rules (LOCKED)
- Every page must have a full-bleed background image
- Only Home uses video background
- All other pages use static background images with subtle motion (parallax/slow zoom/grain), not flashy effects

## 11) Page Architecture (LOCKED)
Pages:
- Home: /pages/index.html
- Browse Catalogue: /pages/browse.html
- Search/Finder: /pages/search.html
- Category: /pages/category.html
- Tool Detail: /pages/tool.html
- About/Methodology: /pages/about.html
- Affiliate Disclosure: /pages/disclosure.html

Home sections:
1) Hero (video background) + headline + subheadline + primary CTA (Browse) + secondary CTA (Find)
2) What this is (brief)
3) Browse by Category
4) Featured/Popular Tools
5) How it Works (3 steps)
6) Final CTA

Browse sections:
1) Page intro (short)
2) Tool grid/list
3) Filters (basic)
4) Pagination/load more

Search/Finder sections:
1) Finder form
2) Results area
3) Refine results option

Category sections:
1) Category header
2) Tool list for category
3) Related categories

Tool Detail sections:
1) Tool name + summary
2) What it does
3) Best use cases
4) Pros/limitations
5) Affiliate CTA (clear + honest)
6) Related tools

About sections:
1) Why this exists
2) How tools are selected
3) Transparency note

Disclosure sections:
1) Affiliate disclosure
2) Editorial independence statement

## 12) Agent Authority Limits (LOCKED)
Agents may:
- Implement the approved structure and sections
- Generate static page background images (only with El’s approval)
- Write copy (only with El’s approval)

Agents may NOT:
- Modify logo files
- Modify hero video
- Invent or change brand rules
- Add new pages beyond the locked list
- Batch changes across multiple sections without approval
- Proceed without explicit approval after each section

## 13) Approval Process (LOCKED)
Work must be delivered one section at a time:
Section → Review → Approval → Lock → Next section
If uncertain: STOP and ask.
If conflict: MCD wins.
